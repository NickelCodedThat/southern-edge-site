# Southern Edge Landscaping

A custom, responsive single-page website for Southern Edge Landscaping, serving Horry County, South Carolina.

## Preview locally

Open `index.html` directly in a browser, or serve the folder with any simple local web server.

## Site files

- `index.html` — page structure, content, metadata, and accessible markup
- `styles.css` — responsive design, layout, and animation
- `script.js` — mobile navigation, media controls, lazy video loading, reveal effects, and estimate-form behavior
- `img/optimized/` — lighter display copies and video poster images; original photos remain untouched
- `vid/optimized/` — smaller web-ready video copies; original videos remain untouched

## Before launch

1. Replace every `[ ADD BUSINESS PHONE ]` placeholder in `index.html`.
2. Replace every `[ ADD BUSINESS EMAIL ]` placeholder in `index.html`.
3. Add the real email to `businessEmail` near the bottom of `script.js` so estimate requests open a prefilled email.
4. Add the final production website URL to the social metadata and structured data if desired.

No third-party JavaScript or CSS dependencies are required.
